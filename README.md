# 🤖 AI Chatbot using Machine Learning

## 📌 Overview
This project is an AI-powered chatbot built using Machine Learning techniques.  
It uses **TF-IDF Vectorization** and **Logistic Regression** to classify user intent and generate appropriate responses.

The chatbot is deployed using **Streamlit**, providing an interactive chat-based user interface.

---

## 🚀 Features
- 💬 Chat-based UI using Streamlit
- 🧠 Intent classification using Machine Learning
- 🔍 TF-IDF for text feature extraction
- ⚡ Logistic Regression model for prediction
- 📊 Confidence-based response handling
- 🔁 Dynamic and combined responses
- 🗂 Session-based chat history

---

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Streamlit
- NLTK

---

## ⚙️ How It Works
1. User inputs a query
2. Text is preprocessed (lowercasing)
3. TF-IDF converts text into numerical features
4. Logistic Regression predicts the intent
5. Chatbot returns a relevant response
6. If confidence is low → fallback message
